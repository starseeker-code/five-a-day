Datos de prueba

Llena la base de datos con datos realistas para probar todas las funcionalidades

El comando de siembra crea un conjunto completo de datos de prueba para ejercitar cada funcionalidad sin tener que ingresar datos manualmente:

  • 3 profesores y 5 grupos (Starters A/B, Movers, Flyers, Adultos)
  • 6 padres con DNI, telefono, email e IBAN
  • 12 alumnos vinculados a padres, asignados a grupos
  • 3 adultos con sus propios datos de contacto
  • 1 alumno inactivo (baja — para probar filtros)
  • Matriculas activas con planes de pago mensual y trimestral
  • Pagos en distintos estados: completados, pendientes y vencidos
  • Horarios, tareas pendientes y entradas en el historial
